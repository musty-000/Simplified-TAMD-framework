# TAMD Fashion Inventory Management System

## A Simplified TAMD Framework for Seasonal Inventory Management in Fashion SMEs

### 📖 Overview

This system implements a simplified **Time Series Adaptive Mode Decomposition (TAMD)** framework for fashion SME inventory management. It generates simulated fashion sales data, analyzes seasonal patterns, and provides actionable inventory recommendations.

### 🎯 Features

- **5 Product Groups**: Summer Wear, Winter Wear, All-Year Basics, Festive Wear, Accessories
- **Intelligent Product Grouping**: Similar products grouped for better forecasting
- **TAMD Analysis**: Decomposes sales into Trend, Seasonal, and Residual components
- **Adaptive Forecasting**: Different methods based on complexity
- **Inventory Recommendations**: Safety Stock, Reorder Point, Order Quantities
- **Interactive Dashboard**: User-friendly Streamlit interface
